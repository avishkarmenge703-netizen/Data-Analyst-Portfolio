# Avishkar Menge – Data Analytics Portfolio

This repository contains three complete data projects and a personal portfolio website.

## Projects

1. **Interactive Dashboard (Streamlit)** – Live dashboard for sales analysis.  
2. **Automated Reporting Script** – Python script that generates a PDF report.  
3. **“From Mess to Message” Case Study** – Data cleaning and analysis of messy data.

## How to Run Each Project

### Project 1: Dashboard
- Install requirements: `pip install -r project1_dashboard/requirements.txt`
- Run: `streamlit run project1_dashboard/app.py`
- Deploy on Streamlit Cloud for a live link.

### Project 2: Reporting
- Install requirements: `pip install -r project2_reporting/requirements.txt`
- Run the Jupyter notebook or the Python script. PDF will be generated.

### Project 3: Case Study
- First run `generate_messy_data.py` to create the messy CSV.
- Then open `clean_and_analyze.ipynb` in Jupyter or Google Colab.

### Portfolio Website
Open `portfolio_website/index.html` in any browser, or host it on GitHub Pages / Netlify.
