# Automated Sales Reporting Script

This script reads sales data, cleans it, generates top customers and seasonal trends, and exports a polished PDF report.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Place `sample_data.csv` in the same folder.
3. Run the Jupyter notebook or the Python script.
4. The output `sales_report.pdf` will be created.
