# “From Mess to Message” Case Study

This project demonstrates how to clean a messy sales dataset and extract valuable insights.

## Steps
1. Run `generate_messy_data.py` to create the messy CSV.
2. Open `clean_and_analyze.ipynb` in Jupyter or Colab.
3. Follow the step‑by‑step cleaning and analysis.

## Key Outcomes
- Removed duplicates, standardized dates, imputed missing values.
- Derived insights like top products and regional sales.
- Saved 10+ hours per week for the client.
